About Me

I believe that forward speaking and clear communication lead to better relationships in life. I hold the same for my web development. When people use my product I feel that my users are in part forming a relationship with me, so I like it to be clean, clear and functional.

I come from a family of carpenters, so in my free time I love to build and work with wood. I can't wait for the day when I have my own woodshop, that I can spend all weekend tinkering around in. I get this desire to build from my father and I believe its why I enjoy web development so much.

Latest Projects

Project Brain Storm

A speed typing game made with Phaser.js

New

Project Brain Storm is a fast paced marathon style typing game built using the phaser.js library. I'm pretty pumped that i was able to add some sweet tunes and rad animations. Next i would like to tailor it to mobile and make it a speed texting game.

juniorDev

A job board built for junior level developers

New

juniorDev is a simple job board pulling in web development jobs with the search terms "junior" and "entrylevel" from the dice api. You can filter the results by entering a zipcode for the area you are searching in.

Yammerings on Twitter

A group of graphs built with D3.js and consuming twitters streaming api

New

Yammerings on Twitter is our consumption and vizualization on the data we gathered from the twitter streaming api. We used D3.js to build our graphs and we built the backend using ruby on rails. We deployed the app using Heroku, and currently there is still work to be done.
